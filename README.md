# PyLiMan

**Python Library Manager** — графическое приложение для управления всеми библиотеками Python.  
Все библиотеки берутся с [PyPI](https://pypi.org/).  

---

## Возможности

- Установка любых библиотек Python из официального PyPI
- Обновление и удаление установленных библиотек
- Поиск библиотек по имени
- Просмотр уже установленных библиотек
- Прогресс установки с полосой и процентами
- Тёмная и светлая тема с сохранением выбранной настройки
- Полностью автономный `.exe` для Windows

---

## Установка и запуск

1. Скачайте **PyLiMan.exe** из [релиза](https://github.com/staryi441/PyLiMan/releases).  
2. Запустите приложение — оно сразу проверит установленные библиотеки и будет готово к работе.  
3. Вкладка «Установленные» показывает все библиотеки.  
4. Чтобы обновить или удалить библиотеку, выберите её из списка и нажмите нужную кнопку.  

---

## Требования

- Windows 10 или выше (64-бит)
- .NET 8 встроен в сам exe → установка .NET на ПК не требуется
- Python 3.x должен быть установлен на компьютере
